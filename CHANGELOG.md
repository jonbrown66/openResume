# Changelog

All notable changes to openResume will be documented in this file.

This project follows a simple date-based changelog until stable versioning is introduced.

## Unreleased

- Added repository community files for issues, pull requests, contribution guidance, security reporting, and CI.
- Improved GitHub-facing README metadata and discoverability language.

## 0.1.0 - Initial Public Preview

- Local-first resume workspace with Markdown and block editing.
- A4 live preview with template and style controls.
- Import support for Markdown, text, PDF, and DOCX resumes.
- AI-assisted resume editing with before/after diff preview.
- Export support for PDF, DOCX, and HTML.
- Local multi-project resume management.
